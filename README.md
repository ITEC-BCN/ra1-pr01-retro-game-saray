# Flappy Bubble

## Integrantes
- Saray Varela  

## Lore del Juego
**Flappy Bubble** sigue la historia de **Bubble**, un slime proveniente del reino **Bultos** de la serie “Hora de aventuras”.  
Bubble viaja accidentalmente en una nave y aterriza en la Tierra, concretamente en **New York (EE.UU.)**.  

El jugador debe ayudar a Bubble a sobrevivir evitando obstáculos que aparecen como tuberías flotantes.  
Si Bubble cae al suelo o toca algún obstáculo, el juego termina.

## Personajes
- **Bubble**: El jugador principal, un slime con habilidades de salto.  
- **Obstáculos (tuberías)**: Aparecen arriba y abajo con un hueco para que Bubble pueda pasar.  

## Gameplay
- Presiona la tecla **Up** para que Bubble salte y se mantenga en el aire.  
- Evita tocar las tuberías o el suelo.  
- Cada conjunto de tuberías superado incrementa tu puntuación.  
- El juego termina si Bubble toca un obstáculo o el suelo.

## Funcionalidades
- Menú principal interactivo con transición de pantalla.  
- Sistema de puntuación visible en pantalla.  
- Obstáculos generados aleatoriamente con huecos de tamaño fijo.  
- Efecto de sonido al saltar.  
- Uso de listas para manejar los obstáculos de forma modular y reutilizable.  

## Assets
- `bubble` → sprite del jugador  
- `pipe_top` y `pipe_bottom` → sprites de los obstáculos  
- `start background` y `game_fondo` → fondos del juego  
- `start_song` → música de inicio
